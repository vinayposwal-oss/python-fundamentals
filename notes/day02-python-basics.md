## What are Variables?

A variable is a named container used to store data. It allows us to save information and use it later in our program.

Example:

```python
name = "Vinay"
age = 20
```

---

## Data Types

Python stores different kinds of data using different data types.

| Data Type | Example | Description |
|-----------|---------|-------------|
| `str` | `"Hello"` | Text or string |
| `int` | `25` | Whole numbers |
| `float` | `5.8` | Decimal numbers |
| `bool` | `True` | True or False values |

---

## print()

The `print()` function displays output on the screen.

Example:

```python
print("Hello World")
```

Output:

```
Hello World
```

---

## input()

The `input()` function takes input from the user.

Example:

```python
name = input("Enter your name: ")
```

---

## type()

The `type()` function tells us the data type of a variable.

Example:

```python
age = 20
print(type(age))
```

Output:

```
<class 'int'>
```

---

## Key Points

- Variables store data.
- Python has different data types for different kinds of information.
- `print()` is used to display output.
- `input()` is used to receive user input.
- `type()` helps identify the type of data stored in a variable.
