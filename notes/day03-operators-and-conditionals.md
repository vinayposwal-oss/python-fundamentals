# Day 3 - Operators & Conditional Statements

## Topics Covered

- Arithmetic Operators
- Comparison Operators
- Logical Operators
- if, elif, else

## Key Learnings

- Arithmetic operators perform mathematical calculations.
- Comparison operators compare values and return `True` or `False`.
- Logical operators combine multiple conditions.
- Conditional statements help programs make decisions based on conditions.

## Reflection

Today I learned how Python makes decisions using conditional statements. These concepts are essential for writing interactive and intelligent programs.
